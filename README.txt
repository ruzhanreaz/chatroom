SIMPLE CHAT APPLICATION
======================

HOW TO RUN:
-----------
1. Double-click "RUN.bat"
2. Wait for "Started DemoApplication" message
3. Open browser to: http://localhost:8080/ws-client.html

USAGE:
------
1. Enter your name and click "Join"
2. Type messages and click "Send" (or press Enter)
3. Open multiple browser tabs to chat with yourself
4. Click "Leave" to disconnect

FEATURES:
---------
- Real-time messaging
- See who's online
- Join/leave notifications
- Message history stored in database
- Simple, clean interface

TO STOP:
--------
Press Ctrl+C in the command window
